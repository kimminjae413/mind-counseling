# 마인드케어 (Mind Counseling)

AI와 전문 상담사가 함께하는 마음 관리 플랫폼

## 주요 기능

- **AI 마음 상담** — 24시간 공감 기반 대화 상담
- **전문 상담 예약** — 검증된 상담사와 1:1 상담 (화상/음성/채팅)
- **심리검사** — PHQ-9 우울, GAD-7 불안 자가진단
- **감정 체크인** — 매일의 기분 기록과 패턴 분석

## 기술 스택

- HTML / CSS (Tailwind CDN) / Vanilla JavaScript
- Pretendard 웹폰트 / Iconify Solar 아이콘
- LocalStorage 기반 데이터 저장
- GitHub Pages 배포

## 실행 방법

`index.html`을 브라우저에서 열면 바로 실행됩니다.
